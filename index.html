• git clone - клонувати репозиторій
  • git add . - додати змінені файли до проекту
  • git status - перевірити статус завантажених файлів та чи є зміни
  • git log - переглянути попередні коміти
  • git commit -m ""- створити коміт
  • git commit -am ""- поєднання команд add та commit
  • git push origin main - відправити зміни до проекту на сервер ГітХаб
  • git push origin HEAD - відправити зміни до обраної гілки проекту в ГітХаб
  • git checkout -b NAME - створити та перейти на нову гілку
